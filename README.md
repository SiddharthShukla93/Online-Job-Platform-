# Online Job Platform (Mini Project)

A web-based mini project to simplify the job search process using only HTML, CSS, and JavaScript.

## Features

- Role-based registration (Job Seeker / Employer)
- Role-based login (Job Seeker / Employer)
- Logout button with session indicator
- Profile creation and update
- Interactive job listing display
- Search and filter by location/type
- Save jobs locally
- Employer panel to post, edit, and delete jobs
- Responsive interface for desktop and mobile

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage for simple data persistence

## How to Run

1. Open `index.html` in any modern web browser.
2. Register a new account and choose a role:
	- Job Seeker: access profile + apply/save jobs.
	- Employer: access employer panel to post/manage jobs.
3. Login with the same role you used during registration.
4. Browse jobs, update profile (seeker), or manage postings (employer).

## Notes

- This is a front-end mini project;
- Data is stored in browser localStorage.
